## Installation

Follow this simple steps to install and run the project

- sudo apt-get update
- sudo apt-get install python3-pip python3-dev
- sudo pip3 install virtualenv
- Clone the project and cd to the dir
- virtualenv BES
- source BES/bin/activate
- pip install flask
- run api.v1.app
