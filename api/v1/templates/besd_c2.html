Exercice :

  1. Perform the stack evaluation of this Bitcoin script and indicate whether it will result in true or false: hex: '010101029301038801027693010487'.
    Answer in file Bitdev/chapter2/answer1


  2. Write a Python function (def stackeval(hex)) that evaluates a stack and returns the result.
    Answer in file Bitdev/chapter2/answer2.py

  3. Given the string 'Btrust Builders', whose bytes encoding is '427472757374204275696c64657273', which is our preimage, generate the redeem script in hex format for the given preimage. Note: redeem_script == OP_SHA256 OP_EQUAL.
    Answer in file Bitdev/chapter2/answer3

  4. Derive an address from the above (2) redeemscript and construct a transaction that sends Bitcoins to the address. Write the transaction hash in the answer file.
    Answer in file bitdev/chapter2/answer4
