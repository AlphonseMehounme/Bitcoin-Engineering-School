Requirements :

Create a repository named Bitdev on github In the repo, create a folder named chapter1 In the folder create the file answer1, answer2, etc for each answer to the exercises.

Exercice :

  1. Perform the stack evaluation of this Bitcoin script and indicate whether it will result in true or false: hex: '010101029301038801027693010487'. Your solution should output the op_codes and/or values.
  2. Given the string 'Btrust Builders', whose bytes encoding is '427472757374204275696c64657273', which is our preimage, generate the redeem script in hex format for the given preimage. Note: redeem_script => OP_SHA256 OP_EQUAL
  3. Derive an address with from the above (2) redeemscript and construct a transaction that send Bitcoins to the address
  4. Construct another transaction that spends from the above (3) transaction given that you have both locking and unlock scripts. Ideally, this would mean you write a transaction builder that constructs the transaction and outputs the transaction ID. Broadcast the transaction on you local bitcoin network via the CLI.
